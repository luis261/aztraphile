# aztraphile

## Introduction
aztraphile is an automation tool intended to help you accelerate the process of rolling out new Python Azure Functions as well as ensuring you are adequately equipped to keep those Functions running smoothly. Overall, this entails:
- automated resource provisioning in Azure
- pre-written code (and unittest) samples using the brand new V2 programming model for Python Azure Functions
- automated setup of Azure DevOps constructs that assure quality while conducting continuous deployments
- flexible monitoring and alerting capabilities designed to fit your needs
